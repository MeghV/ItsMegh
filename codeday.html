Welcome to Game+=code

Looking for a quick project to learn html5 canvas and javascript?

Game+=code is a collaborative game community of ten second games produced by new developers like you!

We provide you with a 500x500 canvas with the id of "game". We support jQuery and Create.js, and will add more libraries upon request. The score is stored in a global variable called score, so just type "score++;" when the player wins to update their score.

Just write a .js file and submit it to be added to the collection.

More documentation to come soon.
